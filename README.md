# 📘 Programação Estruturada em C

Repositório com o material de estudo do curso de **Programação em C**, organizado por módulos. Cada aula possui (ou terá) uma página própria com slides e conteúdo.

> Páginas hospedadas via **GitHub Pages**.

---

## 📑 Índice de Aulas

### Módulo 1 — Fundamentos
1. Primeiros Passos com a Programação em C
2. Variáveis e Saída de Dados (Print)
3. Tipos de Dados em C
4. Lendo Entrada do Usuário
5. Comentários em C

### Módulo 2 — Operadores e Conversões
6. Operadores em C
7. Conversão de Tipos (Implícita e Explícita)
8. Operadores Booleanos e de Comparação

### Módulo 3 — Estruturas de Controle
9. Estruturas If / Else
10. Operador Ternário
11. Estrutura Switch

### Módulo 4 — Laços de Repetição
12. Laço `while`
13. Laço `for`
14. `break` e `continue`

### Módulo 5 — Funções
15. Funções em C
16. Escopo de Variáveis
17. Funções da Biblioteca Padrão
18. [Recursão](https://francoisbarbosa-wq.github.io/programacao_estruturada/slides/recurcao.html) ✅

### Módulo 6 — Vetores e Strings
19. Vetores (Arrays)
20. Vetores Multidimensionais
21. Strings em C
22. Funções de Strings

### Módulo 7 — Ponteiros
23. Ponteiros em C
24. Ponteiros e Vetores
25. Ponteiros e Funções

### Módulo 8 — Estruturas de Dados e Memória
26. Structs
27. Enumerações (enums)
28. Alocação Dinâmica de Memória

### Módulo 9 — Tópicos Avançados
29. Manipulação de Arquivos
30. Pré-processador e Macros
