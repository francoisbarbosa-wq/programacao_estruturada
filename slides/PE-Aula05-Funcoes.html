Funções

O que são?

As funções são blocos de código que executam uma tarefa específica e podem ser chamadas em diferentes partes de um programa.

  • OrganizaçãoSeparam o programa em partes lógicas e legíveis.
  • ModularidadeCada função resolve um problema bem definido.
  • ReutilizaçãoEscreve-se uma vez e usa-se quantas vezes for preciso.

Funções

Sintaxe

Toda função tem quatro partes: o tipo de retorno, o nome, a lista de parâmetros e o corpo.

A assinatura da função é: tipo de retorno + nome + parâmetros.
tipo nome(tipo a, tipo b) { // corpo da função return valor; }

Funções

Declaração (protótipo)

Especifica o nome, o tipo de retorno e os parâmetros — ou seja, a assinatura.

É colocada antes da função main().
// protótipo (antes da main) int soma(int a, int b); int main(void) { // ... return 0; }

Funções

Definição

Contém o corpo da função, onde a lógica é implementada.

Pode vir antes ou depois da main() (se houver protótipo).
int soma(int a, int b) { int resultado; resultado = a + b; return resultado; }

Funções

Chamada

Local onde a função é invocada. Os argumentos são passados e a execução continua após o retorno.

O valor retornado pode ser guardado ou usado diretamente.
int main(void) { int x; x = soma(3, 4); printf("%d\n", x); return 0; }

Parâmetros & Argumentos

  • ParâmetrosVariáveis declaradas na definição da função.
  • ArgumentosValores reais fornecidos na chamada.
// a e b são PARÂMETROS int soma(int a, int b) { return a + b; } // 3 e 4 são ARGUMENTOS x = soma(3, 4);

Retorno (return)

A função pode devolver um valor com a palavra-chave return, transmitindo o resultado de volta ao local de chamada.

O tipo do valor retornado deve coincidir com o tipo de retorno declarado.
float media(float a, float b) { return (a + b) / 2; }

Funções

Sem retorno (void)

Uma função sem retorno (procedimento) executa uma tarefa, mas não devolve valor.

Útil para ações que não exigem um resultado — como imprimir na tela.
void saudacao(char nome[]) { printf("Olá, %s!\n", nome); }

Exercícios

  1. Função que recebe um número inteiro e devolve o seu dobro.
  2. Função que verifica se um número é positivo ou negativo: retorna 1 (positivo), -1 (negativo) e 0 (se igual a 0).
  3. Função + programa de teste para o volume de uma esfera (raio por parâmetro). V = 4/3 · π · r³
  4. Função que recebe dois números e retorna qual deles é o maior.
  5. Função que recebe altura e raio de um cilindro e retorna o seu volume.
  6. Função que recebe três notas e uma letra: A → média aritmética; P → média ponderada (pesos 5, 3 e 2).

Recursão

Uma função recursiva chama a si mesma para resolver um problema em partes menores.

  • Caso baseCondição que encerra a recursão.
  • Caso recursivoA função chama a si própria.
// caso base encerra a recursão int fatorial(int n) { if (n <= 1) return 1; return n * fatorial(n - 1); }
