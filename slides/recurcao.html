RECURSÃO

Chamar a si mesma.

Técnica em que a solução de um problema depende da solução de versões menores do mesmo problema.

O que é?

Uma função recursiva resolve um problema chamando a si mesma para resolver uma versão menor dele.

O salto de fé: defino a solução supondo que a versão menor já está resolvida.
// soma chama a si mesma com n-1 int soma(int n) { if (n == 0) return 0; return n + soma(n - 1); }

Os dois ingredientes

  • Caso baseCondição de parada — responde direto, sem se chamar.
  • Caso recursivoA função se chama com um argumento menor.
Sem caso base, a recursão nunca para → estouro de pilha (stack overflow).
int fatorial(int n) { if (n == 0) return 1; // caso base return n * fatorial(n - 1); // recursivo }

A pilha — fatorial(3)

// IDA — empilhando fatorial(3) → 3 * fatorial(2) fatorial(2) → 2 * fatorial(1) fatorial(1) → 1 * fatorial(0) fatorial(0) → return 1 // caso base // VOLTA — desempilhando (o cálculo acontece aqui) fatorial(0) = 1 fatorial(1) = 1 * 1 = 1 fatorial(2) = 2 * 1 = 2 fatorial(3) = 3 * 2 = 6

Com vs. sem recursão

Fatorial de n

Com recursão
int fatorial(int n) { if (n == 0) return 1; return n * fatorial(n - 1); }
Sem recursão (laço)
int fatorial(int n) { int resultado = 1; for (int i = 1; i <= n; i++) resultado = resultado * i; return resultado; }

Potência (base^exp)

Com recursão
int potencia(int base, int exp) { if (exp == 0) return 1; return base * potencia(base, exp - 1); }
Sem recursão (laço)
int potencia(int base, int exp) { int resultado = 1; for (int i = 0; i < exp; i++) resultado = resultado * base; return resultado; }

Com vs. sem recursão

Contar de n até 1

Com recursão
void contar(int n) { if (n == 0) return; printf("%d ", n); contar(n - 1); }
Sem recursão (laço)
void contar(int n) { for (int i = n; i >= 1; i--) printf("%d ", i); }

O contraste de ouro (ida × volta)

printf na IDA
void conta(int n) { if (n == 0) return; printf("%d ", n); conta(n - 1); } // conta(5) → 5 4 3 2 1
printf na VOLTA
void conta(int n) { if (n == 0) return; conta(n - 1); printf("%d ", n); } // conta(5) → 1 2 3 4 5
O printf da volta fica pendurado esperando a chamada de baixo terminar — a pilha guarda os n na ordem reversa.

Tipos de recursão

  • DiretaA função chama a si mesma. (todos os exemplos até aqui)
  • IndiretaA chama B, que chama A — recursão não é só a função literal se chamar.
  • De cauda (tail)A chamada recursiva é a última operação; alguns compiladores otimizam para um laço.
