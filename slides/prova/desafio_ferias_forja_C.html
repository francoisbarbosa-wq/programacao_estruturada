A Metodologia

Três cadências que trabalham juntas

Os 15 dias não são 15 exercícios soltos: são uma coisa só que evolui. A progressão vem de três camadas simultâneas.

01 · ritmo

Ritmo diário fixo

Todo dia segue os mesmos quatro tempos. O aluno internaliza o ritmo e sabe o que esperar.

02 · níveis

Bronze · Prata · Ouro

Cada desafio tem três degraus. Quem tem dificuldade e quem é avançado têm para onde ir.

03 · fio condutor

Um sistema que cresce

Um único sistema, a Forja, evolui do Dia 1 ao 15, sempre usando o conteúdo do dia.

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>

int main() {
    int xp;
    printf("XP do aventureiro: ");
    scanf("%d", &xp);

    if (xp < 100)                       // do menor para o maior
        printf("Ordem: Bronze\n");
    else if (xp < 300)
        printf("Ordem: Prata\n");
    else if (xp < 600)

  • Encadeie do menor para o maior: o primeiro que casar vence.
  • O else final é a faixa mais alta (rede de segurança).

Decisão Cap. 3 · O Balcão de Missões ◆ Fecha bloco Parte 1 · o modelo

Seleção múltipla: switch-case

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>

int main() {
    int opcao;
    printf("1-Recrutar  2-Consultar  3-Sair\n");
    printf("Escolha: ");
    scanf("%d", &opcao);

    switch (opcao) {
        case 1: printf("Recrutando...\n");  break;
        case 2: printf("Consultando...\n"); break;
        case 3: printf("Ate logo!\n");      break;
        default: printf("Opcao invalida.\n");
    }
    return 0;
}
Aquecimento · retomada

Converter um if-else if de ontem em switch equivalente.

Pontos-chave · o que observar
  • Cada case precisa do break: sem ele, “vaza” para o próximo.
  • O default trata qualquer opção inesperada.

Repetição com while

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>

int main() {
    int xp, total = 0;

    printf("XP (0 encerra): ");
    scanf("%d", &xp);
    while (xp != 0) {                   // 0 e a sentinela
        total += xp;
        printf("XP (0 encerra): ");
        scanf("%d", &xp);
    }
    printf("XP total: %d\n", total);
    return 0;
}
Aquecimento · retomada

Contar de 1 até N com while, imprimindo cada valor.

Pontos-chave · o que observar
  • A sentinela (0) é o valor que encerra o laço.
  • Leia uma vez antes e de novo dentro do laço (padrão “ler-processar”).

Repetição com do-while

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>

int main() {
    int opcao;
    do {                               // executa ao menos 1 vez
        printf("\n1-Recrutar 2-Consultar 3-Sair\n");
        scanf("%d", &opcao);

        if (opcao == 1) printf("Recrutando...\n");
        else if (opcao == 2) printf("Consultando...\n");
    } while (opcao != 3);              // condicao no fim
    printf("Forja fechada.\n");
    return 0;
}
Aquecimento · retomada

Pedir um número de 1 a 3, repetindo até a entrada ser válida.

Pontos-chave · o que observar
  • do-while executa o corpo ao menos uma vez.
  • A condição é testada no fim: ideal para menus.

Estruturas de dados Cap. 9 · Os Feitos da Forja ◆ Fecha bloco Parte 1 · o modelo

Algoritmos com vetores: soma, média, extremos, busca

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>

int main() {
    int xp[] = {120, 340, 90, 500, 210};
    int n = 5, soma = 0;
    int maior = xp[0], menor = xp[0];

    for (int i = 0; i < n; i++) {      // um unico percurso
        soma += xp[i];
        if (xp[i] > maior) maior = xp[i];
        if (xp[i] < menor) menor = xp[i];
    }
    printf("Media: %.1f\n", (float) soma / n);
    printf("Maior: %d  Menor: %d\n", maior, menor);
    return 0;
}
Aquecimento · retomada

Somar todos os elementos de um vetor.

Pontos-chave · o que observar
  • Um único percurso já resolve soma, maior e menor.
  • Inicialize maior/menor com o primeiro elemento, não com 0.

Modularização Cap. 13 · A Magia da Transformação Parte 1 · o modelo

Funções + modularização: retorno de valores e funções sobre vetores

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>

int subirNivel(int nivel) {   // recebe o valor, devolve o novo
    return nivel + 1;
}

int somaXP(int xp[], int n) {   // percorre o vetor inteiro
    int total = 0;
    for (int i = 0; i < n; i++)
        total += xp[i];
    return total;
}

int main() {
    int nivel = 4;
    nivel = subirNivel(nivel);        // guarda o retorno
    printf("Nivel: %d\n", nivel);
    return 0;
}
Aquecimento · retomada

Structs: registros

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>

struct Aventureiro {                   // agrupa varios campos
    char nome[50];
    char classe[20];
    int  nivel;
    int  xp;
};

int main() {
    struct Aventureiro a = {"Aria", "Bardo", 13, 820};
    printf("%s, %s Nv.%d (%d XP)\n",   // acesso com ponto
           a.nome, a.classe, a.nivel, a.xp);
    return 0;
}

Pontos-chave · o que observar
  • A struct agrupa campos de tipos diferentes sob um nome.
  • Acesse cada campo com o operador ponto (a.nivel).

Structs Cap. 15 · A Forja Completa ✦ Integrador Final Parte 1 · o modelo

Vetor de structs: desafio integrador final

modelo guiado · leia antes de escrever forja.c 
#include <stdio.h>
#define MAX 100

struct Aventureiro { char nome[50]; int nivel, xp; };

int main() {
    struct Aventureiro forja[MAX];
    int n;
    scanf("%d", &n);                        // quantos herois

    for (int i = 0; i < n; i++)             // cadastro
        scanf("%49s %d %d", forja[i].nome,
              &forja[i].nivel, &forja[i].xp);

    for (int i = 0; i < n; i++)             // listagem
        printf("%s Nv.%d - %d XP\n", forja[i].nome,
               forja[i].nivel, forja[i].xp);
    return 0;
}
Aquecimento · retomada

Declarar um vetor de structs de tamanho MAX.

Pontos-chave · o que observar
  • Vetor de structs = várias fichas: forja[i] escolhe o herói.
  • O índice escolhe o aventureiro; o ponto escolhe o campo.
