Decisão Cap. 3 · O Menu de Golpes ◆ Fecha bloco Parte 1 · o modelo

Seleção múltipla: switch-case

modelo guiado · leia antes de escrever guerreiros.c 
#include <stdio.h>

int main() {
    int opcao;
    printf("1-Treinar  2-Consultar  3-Sair\n");
    printf("Escolha: ");
    scanf("%d", &opcao);

    switch (opcao) {
        case 1: printf("Treinando...\n");  break;
        case 2: printf("Consultando...\n"); break;
        case 3: printf("Ate logo!\n");      break;
        default: printf("Opcao invalida.\n");
    }
    return 0;
}
Aquecimento · retomada

Converter um if-else if de ontem em switch equivalente.

Pontos-chave · o que observar
  • Cada case precisa do break: sem ele, “vaza” para o próximo.
  • O default trata qualquer opção inesperada.

Repetição com while

modelo guiado · leia antes de escrever guerreiros.c 
#include <stdio.h>

int main() {
    int ki, total = 0;

    printf("Ki (0 encerra): ");
    scanf("%d", &ki);
    while (ki != 0) {                   // 0 e a sentinela
        total += ki;
        printf("Ki (0 encerra): ");
        scanf("%d", &ki);
    }
    printf("Ki total: %d\n", total);
    return 0;
}
Aquecimento · retomada

Contar de 1 até N com while, imprimindo cada valor.

Pontos-chave · o que observar
  • A sentinela (0) é o valor que encerra o laço.
  • Leia uma vez antes e de novo dentro do laço (padrão “ler-processar”).

Repetição com do-while

modelo guiado · leia antes de escrever guerreiros.c 
#include <stdio.h>

int main() {
    int opcao;
    do {                               // executa ao menos 1 vez
        printf("\n1-Treinar 2-Consultar 3-Sair\n");
        scanf("%d", &opcao);

        if (opcao == 1) printf("Treinando...\n");
        else if (opcao == 2) printf("Consultando...\n");

}

  • do-while executa o corpo ao menos uma vez.
  • A condição é testada no fim: ideal para menus.

#include <stdio.h>

int main() {
    int n, dano, total = 0;

Estruturas de dados Cap. 9 · A Leitura do Scouter ◆ Fecha bloco Parte 1 · o modelo

Algoritmos com vetores: soma, média, extremos, busca

modelo guiado · leia antes de escrever guerreiros.c 
#include <stdio.h>

int main() {
    int ki[] = {120, 340, 90, 500, 210};
    int n = 5, soma = 0;
    int maior = ki[0], menor = ki[0];

    for (int i = 0; i < n; i++) {      // um unico percurso
        soma += ki[i];
        if (ki[i] > maior) maior = ki[i];
        if (ki[i] < menor) menor = ki[i];
    }
    printf("Media: %.1f\n", (float) soma / n);
    printf("Maior: %d  Menor: %d\n", maior, menor);
    return 0;
}
Aquecimento · retomada

Somar todos os elementos de um vetor.

Pontos-chave · o que observar
  • Um único percurso já resolve soma, maior e menor.
  • Inicialize maior/menor com o primeiro elemento, não com 0.

Vetor de structs: desafio integrador final

modelo guiado · leia antes de escrever guerreiros.c 
#include <stdio.h>
#define MAX 100

struct Guerreiro { char nome[50]; int nivel, ki; };

int main() {
    struct Guerreiro equipe[MAX];
    int n;
    scanf("%d", &n);                        // quantos guerreiros

    for (int i = 0; i < n; i++)             // cadastro
        scanf("%49s %d %d", equipe[i].nome,
              &equipe[i].nivel, &equipe[i].ki);

    for (int i = 0; i < n; i++)             // listagem
        printf("%s Nv.%d - %d ki\n", equipe[i].nome,
               equipe[i].nivel, equipe[i].ki);
    return 0;
}
Aquecimento · retomada

Declarar um vetor de structs de tamanho MAX.

Pontos-chave · o que observar
  • Vetor de structs = várias fichas: equipe[i] escolhe o guerreiro.
  • O índice escolhe o guerreiro; o ponto escolhe o campo.
